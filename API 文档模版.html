总览

状态码

采用标准的HTTP状态码

状态码 描述
200 OK - 成功
201 Created - 创建成功
204 No Content - 成功但无返回内容
400 Bad Request - 请求错误，一般是请求的参数不正确
401 Unauthorized - 未授权，前端需要重新请求授权
403 Forbidden - 权限不够
404 Not Found
500 Internal Server Error

微信 REDIRECT_URI 回调

非对外使用，此 API 即为 REDIRECT_URI 中填入的地址，在客户端向微信请求到 code 后，微信会回调这些 API，服务端在这里处理 code 和 accessToken。

说明：只要从微信端第一次访问页面，所有信息（除手机号意外）都会被写入数据库生成一条用户信息，即使没有手机注册，下次进入也只会使用之前存储的信息。
